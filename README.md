# Postkoll-loggbok

## vad är projektet
Ge data till en chart som visar hur stor sannolikheten är att man får post varje timme. 

## Första idén 
Första idén var att använda en ultraljudssensor för att med hjälp av ultraljud känna av om det finns post i brevlådan. Problemet som uppstod var att datan vi fick av ultraljuds sensorn var distans i form av centimeter och inches och var inte tillräckligt känslig för att länna av om det låg ett brev i botten eller inte. 

## Andra idén 
Använda magnet sensorer, en i locket av brevlådan och en parallellt i kanten så att magneterna alltid nuddar varandra när lådan är stängd.
